Examples for Use Case: Specify the controls available on creating a map

The map viewers here are displaying the location of CERN, birthplace of the web: 

CERN
Esplanade des Particules 1
1217 Meyrin
Switzerland

In each case, the map was created and initialised using the API provided by the specified library. Where available, the example code provided in the API documentation has been used with minor modifications.

OpenLayers with OpenStreetMap tiles: Default controls for comparison

The default controls for this library are: zoom, attribution, rotate. The last of these can be accessed with Shift-Alt + drag which will cause the control to appear.

OpenLayers with OpenStreetMap tiles: No controls

Note that although the controls such as zoom are not present, the map can still be manipulated via user interactions such as the scroll wheel.

OpenLayers with OpenStreetMap tiles: Zoom control only

Note that although only the zoom control is present, the map can still be manipulated via user interactions such as Shift-Alt + drag.
